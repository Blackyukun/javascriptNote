## 语法

JavaScript的每个语句以`;`结束，虽然JavaScript不强制，但是防止运行结果与期望不一致，我们都要加`;`.语句块用`{...}`，语句块里面可以嵌套使用`{...}`。

JavaScript语法严格区分大小写，如果弄错，程序会报错或运行不正常。

```JavaScript
//以这个开头直到行末的字符为注释
/*这是块注释
这里也是
注释结束*/
var x = 1; //完整的赋值语句
'Hello, world!'; //一个字符串，也是一个完整的语句
//这是一个语句块，判断语句，判断成立，执行{...}中的语句
if (2 > 1) {
	x = 1;
}
```

## 数据类型

### Number

JavaScript不区分整数和浮点数，统一用Number表示。

Number可以直接做四则运算，规则与数学一致，`%`是求余的运算符号，`10%3;`值为`1`。`2/0;`值为`Infinity`，`0/0;`的值为`NaN`。

```JavaScript
123; // 整数123
0.456; // 浮点数0.456
1.2345e3; // 科学计数法表示1.2345x1000，等同于1234.5
-99; // 负数
NaN; // NaN表示Not a Number，当无法计算结果时用NaN表示
Infinity; // Infinity表示无限大，当数值超过了JavaScript的Number所能表示的最大值时，就表示为Infinity
```

### 字符串

字符串是以英文的单引号或双引号包裹的任意文本，`''`和`""`只是一种表现方式，不是字符串一部分。比如`'abc'`，`"abc"`只有a,b,c这3个字符。

### 布尔值

布尔值经常用于判断语句中，一个布尔值只有`true`,`false`两种值，要么是`true`，要么是`false`，可以直接是这两个值表示布尔值，也可以通过布尔运算计算出来：

`&&`运算是与运算，相当于数学中的“且”，只有所有值为true，`&&`运算结果才是true。

`||`运算是或运算，相当于数学中的“或”，只要其中一个为true，`||`运算结果就是true。

`!`运算是非运算，相当于数学中的“非”，他是一个单目运算符，把true变为false，false变为true。

```JavaScript
true; //这是一个true值
false; // 这是一个false值
2 > 1; //这是一个true值
2 >= 3; //这是一个false值

true && true; // 这个&&语句计算结果为true
true && false; // 这个&&语句计算结果为false
false && true && false; // 这个&&语句计算结果为false

false || false; // 这个||语句计算结果为false
true || false; // 这个||语句计算结果为true
false || true || false; // 这个||语句计算结果为true

! true; // 结果为false
! false; // 结果为true
! (2 > 5); // 结果为true
```
