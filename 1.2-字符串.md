## 字符串

JavaScript的字符串就是用`''`或`""`括起来的字符表示。如果`'`本身也是一个字符，可以用`""`括起来。

`\`为转义字符，比如转义`'`和`"`：
```JavaScript
'I\'m \"OK\"!'; //表示I'm "OK"!
```
`\`可以转义很多字符，比如`\n`表示换行，`\t`表示制表符，`\`转义本身就可以`\\`表示。

## 多行字符串

多行字符串可以使用`\n`，也可以使用反引号表示：
```JavaScript
`这是一个
多行
字符串`; //反引号在键盘ESC下方
```

## 模版字符串

要把多个字符串连接起来，可以用`+`号链接，如果有很多变量要连接，可以使用模版字符串：
```JavaScript
var name = '小明';
var age = 20;
var message = '你好, ' + name + ', 你今年' + age + '岁了!';
var message2 = `你好, ${name}, 你今年${age}岁了!`; //会自动替换字符串中的变量
alert(message);
alert(message2);
```

## 操作字符串

#### 字符串属性

```JavaScript
var s = 'Hello, world!';
s.length; // 13

s[0]; // 'H'，索引从0开始
s[6]; // ' '
s[7]; // 'w'
s[12]; // '!'
s[13]; // undefined 超出范围的索引不会报错，但一律返回undefined

//需要特别注意的是，字符串是不可变的，如果对字符串的某个索引赋值，不会有任何错误，但是，也没有任何效果
var s = 'Test';
s[0] = 'X';
alert(s); // s仍然为'Test'
```
